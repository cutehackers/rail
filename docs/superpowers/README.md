# Historical Planning Notes

Files under `docs/superpowers/` are preserved planning and design records from earlier implementation phases.

Some plans and specs intentionally reference the old Dart runtime, `bin/rail.dart`, `pubspec.yaml`, or pre-release migration assumptions because they describe the repository before the Go product transition was complete.

These documents are historical context, not current release or operator documentation.
For the active product contract, use `README.md`, `docs/ARCHITECTURE.md`, and `docs/releases/`.
